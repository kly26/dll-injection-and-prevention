# DLL Injection and Prevention

## Overview

This project demonstrates both offensive and defensive techniques related to Windows Dynamic Link Libraries (DLLs).

The Red Team section focuses on DLL execution and DLL Injection attacks, while the Blue Team section implements real-time monitoring using Sysmon and PowerShell to detect unauthorized DLL loading.

---

## Objectives

- Understand DLL execution techniques.
- Perform DLL Injection into Windows processes.
- Simulate data exfiltration.
- Detect DLL Injection using Sysmon.
- Build a lightweight SOC monitoring console.
